# Advay Acharya - Personal Portfolio

A sleek, modern, and cyberpunk-themed personal portfolio website built to showcase my skills, projects, and professional experience as a **Cybersecurity Analyst & Developer**.

## 🌐 Live Demo
*https://advayacharya.vercel.app*

## 🚀 Features

- **Cyber/Hacker Aesthetics**: Features a custom boot-sequence loading screen, scramble-text effects, and custom cursors to fit a cybersecurity theme.
- **Interactive 3D Background**: Utilizes `Three.js` to render a dynamic, dotted 3D surface that responds to the environment.
- **Radial Orbital Timeline**: A unique, interactive node-based timeline to explore different projects and their technical details.
- **Responsive Design**: Fully responsive layout ensuring the portfolio functions flawlessly across all desktop, tablet, and mobile devices.
- **Zero-Dependency Core**: Built primarily with Vanilla HTML, CSS, and JavaScript, leveraging only `Three.js` for the 3D canvas rendering.

## 🛠️ Technology Stack

- **HTML5** for semantic structure.
- **CSS3** for styling, animations, and custom visual effects.
- **JavaScript (ES6+)** for DOM manipulation, dynamic text effects, and logic.
- **Three.js** for the 3D interactive background.

## 📁 Project Structure

- `index.html`: The main structural entry point of the application.
- `styles.css`: Contains all custom styling, animations (like the marquee and text scramble), and layout configurations.
- `script.js`: Handles the boot sequence, 3D background rendering logic, custom cursor, and the interactive orbital timeline mechanics.
